## 为什么选择数据分析？

数据驱动的决策正在改变每个行业。从电商用户行为分析到智能物流平台，我逐渐意识到：**数据本身没有价值，价值来自于对数据的分析和洞察**。

## 我的技术栈演进

### 本科阶段（2020-2024）
在电子科技大学成都学院，我打下了扎实的计算机基础：
- C 语言和 Java 编程
- 数据结构与算法
- 数据库系统原理
- 操作系统和计算机网络

### 硕士阶段（2024-至今）
在法国 ESIGELEC，我开始专注于大数据和数据分析：
- Hadoop 生态系统（HDFS, MapReduce, Hive）
- 数据科学与机器学习（Python, Scikit-learn）
- 商业智能与数据分析（Power BI, Tableau, Talend）
- 大规模数据处理（Apache Spark）

## 实战项目经验

### 电商用户行为分析
使用 Python 和 SQL 分析模拟电商数据，计算复购率、客单价等核心指标。这个项目让我真正理解了**数据清洗的重要性**——原始数据往往是混乱的，80%的工作都在清洗和预处理。

### 商业智能（BI）Dashboard
使用 Power BI 和 Talend 构建 ETL 流程和交互式仪表盘。学会了如何将复杂数据转化为直观的可视化展示。

### 智能物流数据分析平台
最复杂的一个项目。使用 RFID 数据，结合 Flask 搭建 Web 平台，实现数据上传、模型预测、结果可视化。这是第一次从数据到部署的完整项目体验。
